Modelagem Formal do Tri√¢ngulo de Ferro#

Dimens√µes Fundamentais:

1. Escopo (S): Funcionalidades e caracter√≠sticas entregues
2. Tempo (T): Cronograma e time-to-market  
3. Custo (C): Recursos financeiros e humanos
4. Qualidade (Q): N√≠vel de qualidade t√©cnica e percebida

Rela√ß√£o de Interdepend√™ncia:

Lei da Conserva√ß√£o de Valor em Projetos:
S √ó Q = f(T, C)

Onde f representa uma fun√ß√£o de produtividade que depende de:
- Maturidade da equipe
- Complexidade t√©cnica
- Press√µes externas
- Ferramentas dispon√≠veis

Modelagem Matem√°tica Simplificada:

Para projeto fixo com recursos limitados:
S + Œ±T + Œ≤C + Œ≥Q = K (constante)

Onde Œ±, Œ≤, Œ≥ s√£o pesos contextuais que variam por:
- Tipo de projeto (startup vs. enterprise)
